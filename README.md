# tesisAnamorpher
Aca se encuentra todo el codigo utilizado en la tesis incluidos los datasets y outputs de sanitizacion

El codigo fue desarrollado en google collab con conexion a mi carpeta de drive donde tenia las imagenes para probar. 
Si desean probar el codigo recomiendo cambiar esta linea en todos los archivos:
BASE        = "/content/drive/MyDrive/Imagenes tesis"
y remplazarlo con el directorio donde guardaron el datset. 

El desarrollo principal que se utilizo para este proyecto se encuentra en la carpeta Pipeline principal, ahi encontraran el codigo del entrenamiento del modelo detector IDONO, el codigo complento del pipeline de una imagen, los outputs que se obtuvieron del pipeline, el dataset utilizado y las imagenes que se usaron para el POC. 

En la carpeta Metodos e informacion adicional se encuentran otros metodos que fueron utiles para el desarrollo y la explicacion, aquellas imagenes que en el escrito se mencionan como "dificiles", resultados de estas imagenes al pasarlas por la etapa de deteccion OCR.
